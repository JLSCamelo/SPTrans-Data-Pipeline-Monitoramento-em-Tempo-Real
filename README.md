# SPTrans-Data-Pipeline-Monitoramento-em-Tempo-Real
Desenvolvimento de um pipeline de dados ponta a ponta para captura, tratamento e persistência de dados geográficos da frota de ônibus de São Paulo (API Olho Vivo). O projeto utiliza Python para ingestão automatizada, MySQL para armazenamento de séries temporais e Power BI para visualização de métricas de mobilidade urbana.
